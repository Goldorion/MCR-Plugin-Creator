# Security Policy

## Guarentee
MCR Plugin Creator will make no attempt to do any of the following:
* Add a malware on your computer
* Store your files stored on your computer
* Steal any information stored in your computer

## Reporting a Vulnerability

If you happen to find a security vulnerability in MCR Plugin Creator , immediately report an issue with the bug fix template.
Issues that have [SECURITY] in their title will be handled as soon as possible after review.

Depending on the severity, it can take upto a few days to fix. 
**MCR Plugin Creator  is not affiliated with MCreator or Pylo. Report vulnerabilities with it only to me**
